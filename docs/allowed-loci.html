List of permitted loci
DAFF Taxonomic assignment workflow

This page lists permitted values for the locus field in the metadata.csv input file. Many of these loci are synonyms for the same target - it does not matter which synonym is specified as the input.

For virus queries and workflow runs using the BOLD database, the locus field is not required. In these cases, you should set this field to NA.

These values describe the default loci.json file that is defined in the workflow. It is possible for administrators to replace this with a custom file. If this might the case, please request a list of allowed loci from your service administrator.
